      Plugin

      class LenskitPlugin
extends java.lang.Object

      Plugin for LensKit evaluations. This sets up the basic infrastructure required for LensKit tasks to work. To use, add the following to your `build.gradle`: ```groovy apply plugin: 'lenskit' ``` This task only sets up infrastructure and configuration defaults. It does *not* create any tasks.
