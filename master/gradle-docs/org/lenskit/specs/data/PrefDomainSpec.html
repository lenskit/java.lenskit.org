          • @javax.annotation.Nonnull public static PrefDomainSpec fromString(@javax.annotation.Nonnull java.lang.String spec)

            Parse a preference domain from a string specification.

            Continuous preference domains are specified as [min, max]; discrete domains as min:max[/prec/. For example, a 0.5-5.0 half-star rating scale is represented as [0.5, 5.0]/0.5.

            throws:
            IllegalArgumentException if spec is not a valid domain specification.
            Returns:
            The preference domain represented by spec.
            Parameters:
            spec - The string specifying the preference domain.
