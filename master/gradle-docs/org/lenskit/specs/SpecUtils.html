        • @javax.annotation.Nullable public static T buildObject(@javax.annotation.Nonnull java.lang.Class<T> type, @javax.annotation.Nullable AbstractSpec spec)

          Build an object from a specification. On its own, this method does very little. It depends on support from objects in order to work. It operates as follows: 1. Load the SpecHandlers using java.util.ServiceLoader. 2. Query each spec handler, in turn, to try to find one that can build an object of type `type` from the spec. 3. If no such handler is found, try to invoke a static `fromSpec(AbstractSpec)` method on `type`. 4. If no such method is found, or it returns `null`, throw NoSpecHandlerFound.

          throws:
          NoSpecHandlerFound if no spec handler or `fromSpec` method can be found.
          Returns:
          The built object. Will be `null` if and only if `spec` is `null`.
          Parameters:
          type - The type of object to build.
          spec - The specification to use.
          - The built object type.
